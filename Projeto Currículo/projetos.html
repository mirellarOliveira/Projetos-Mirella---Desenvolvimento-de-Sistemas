Projeto TaskWeek

O TaskWeek é um projeto desenvolvido como parte do Projeto Integrador Final do 1º semestre do curso técnico em Desenvolvimento de Sistemas. O objetivo foi criar um gerenciador de tarefas simples e eficiente, permitindo ao usuário cadastrar, listar, buscar e remover tarefas para facilitar a organização do dia a dia.

Funcionalidades:

  • Cadastro de novas tarefas
  • Busca por tarefas específicas
  • Listagem completa
  • Remoção de tarefas

    • Ferramentas utilizadas:

  • HTML e CSS para a interface
  • JavaScript para a lógica e interatividade
  • Node.js para execução e manipulação de arquivos

    BrasilShop

    O BrasilShop foi um projeto desenvolvido em grupo, onde recebemos um site já existente e tivemos o desafio de aplicar melhorias visuais e funcionais. O objetivo foi aprimorar a experiência do usuário, deixando a navegação mais intuitiva e o layout mais moderno.

    Principais melhorias:

  • Ajustes no design e paleta de cores
  • Melhoria na organização do conteúdo
  • Otimização da responsividade para dispositivos móveis
  • Ajustes de acessibilidade

    • Ferramentas Utilizadas:

  • HTML e CSS para estrutura e estilização
